Acerca de TaskFlow

TaskFlow es una aplicación para gestionar tareas diarias con facilidad.

  • Agrega tareas con prioridad.
  • Consulta tu lista de tareas.
  • Elimina las tareas completadas.

Esta aplicación fue desarrollada con Flask y SQLite como base de datos.
